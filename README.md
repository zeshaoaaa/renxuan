# renxuan

《人选天选论》决策剖析、自我觉察与复盘 skill。

基于renxuantianxuan.com中的章节以及直播切片的文稿制作而成

## 包含内容

- `SKILL.md`：skill 主说明
- `references/书籍/`：核心原文
- `references/路飞直播回放（切片）-Get笔记导出/`：补充案例与素材

## 用途

适用于以下场景：

- 用《人选天选论》框架分析纠结、拖延、卡住的问题
- 识别贪婪与恐惧
- 做具体决策复盘
- 使用河流世界、硬币世界、七层捞石法来收敛问题
