# renxuan

《人选天选论》决策剖析、自我觉察与复盘 skill 的独立说明。

这个 skill 不是拿来空谈人生观的，而是拿来分析一件具体的卡点：
你现在到底在选什么，你要付什么代价，你真正怕什么，最后可能走到哪里。

## 仓库内容

- `SKILL.md`：skill 主说明与对话规则
- `references/书籍/`：核心原文
- `references/直播切片/路飞直播回放（切片）-Get笔记导出/`：直播切片案例库
- `references/直播切片/自己装逼的路飞-Get笔记导出/`：补充近期表达与案例的直播切片案例库

## 默认使用规则

这个 skill 强调：

1. 先主动搜索 `references/直播切片/` 下的两个目录
2. 先看 `索引.md`
3. 命中后优先读 `总结/`
4. 必要时再下钻 `原文/` 和 `提取报告.md`
5. 再回到 `references/书籍/` 对照概念

也就是说，不是“需要时再去找直播切片”，而是默认先检索直播切片案例。

## 核心概念速览

### 人选

人选，就是你当下这一刻真正做出的选择。

它不是你嘴上说自己想怎样，而是你最后实际往哪边走。
比如你嘴上说想创业，但一直不动，那你现在的人选，可能其实是继续留在原地。

### 祸

祸，不等于灾难。

在这个框架里，祸更接近“代价”。
你每做一个选择，都要付出某种代价：时间、关系、面子、机会、稳定、钱，甚至旧身份。
所以重点不是“我要不要选”，而是“我愿不愿意付这个代价”。

### 天选

天选，就是结果。

但结果并不只由你一个人决定。
你能决定的是人选，结果还会受到外部变量影响，所以天选里永远有你控制不了的部分。

### 河流世界

河流世界，是看自己的框架。

它把人的内在状态看成一条河：

- 水：不断流动的念头
- 河床：天性、经历、性格底色
- 石头：真正改变流向的贪婪与恐惧
- 波纹：借口、理由、自洽、表层情绪

这个概念的重点是：
不要老盯着波纹看，要往下看石头。

### 硬币世界

硬币世界，是看冲突的框架。

它认为贪婪和恐惧不是轮流出现的，而是同时存在的，就像一枚硬币的两面。
你之所以卡住，往往不是因为没有想要的东西，而是“想要”和“害怕”同时都在。

### 大石头和小石头

这是对驱动力的进一步区分：

- 大石头：更明显、更外露，通常是贪婪，比如想赢、想证明自己、想翻身
- 小石头：更隐蔽、更细碎，通常是恐惧，比如怕失败、怕丢脸、怕白做、怕失去关系

很多人不是没有动力，而是小石头越积越多，把河道堵住了。

### 人之路

人之路，就是一个人逐步看清自己、走向觉醒的路径。

在这个框架里，常见有三种方式：

- 捞石头：靠长期记录和复盘慢慢看清自己
- 潜水：靠深度思考、冥想等方式直接往下探
- 改河道：重塑自己过去形成的路径

这个 skill 默认更偏向第一种，也就是慢慢捞石头。

### 天权

天权，就是“除你之外，一起决定结果的现实力量”。

比如时机、环境、别人、资源、规则、结构，这些都算。
所以做分析时，不能把一切都怪自己，也不能把一切都推给命运。
更稳的做法是分清：哪些是你能选的，哪些是天权在起作用。

### 七层捞石法

七层捞石法，是这个 skill 最核心的复盘工具。

它的顺序是：

1. 事实：发生了什么
2. 第一反应：你当时最直接的感觉
3. 我想得到什么：你的贪婪是什么
4. 我在害怕什么：你的恐惧是什么
5. 我找了什么理由：你给自己讲了什么故事
6. 主石头是什么：这次最核心的驱动力是什么
7. 下次怎么选：下次遇到类似场景怎么做

重点不是写得漂亮，而是把石头真的捞出来。

## 这个 skill 适合做什么

- 分析纠结、拖延、卡住的问题
- 复盘某个决定为什么做成这样
- 区分自己到底是想要，还是害怕
- 看清一个选择背后的代价
- 借直播切片案例和原文概念做交叉对照
