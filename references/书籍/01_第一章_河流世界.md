# 第一章：河流世界

## 河流世界

在我的心中，有许多看待世界的方式，我将其称作小世界——你可以把它理解为一种认知框架。

在所有小世界中，建立最早、也对我的人生与日后事业影响最为深远的，便是「 河流世界 」。

## 思想就是一条河

就在那一瞬间，望着眼前的溪流，我忽然陷入一种奇妙的心境，一个念头猛然浮现：

人的思想，其实就像一条河。

不是像，几乎就是。

| 河流元素 | 对应含义 |
| --- | --- |
| 河里的水 | 时刻流动的思想 |
| 水源 | 外界持续涌入的信息 |
| 河床 | 天性、经历、性格底色 |
| 河底的石头 | 藏在内心的 贪婪与恐惧 |
| 河的宽度 | 知识的广度 |
| 水的深度 | 知识的密度 |
| 水流的速度 | 思考的速度 |
| 河岸 | 知道自己不知道的知识边界 |
| 水面的波纹 | 借口、理由、自洽和表层情绪 |

## 思想的容器

人类自诞生以来，便一直在做一件事： 为河水打造容器 。

- 最早的容器，是口耳相传的故事
- 后来，是刻在龟甲、竹简上的文字
- 再往后，是装订成册的书
- 而后，是画作、音乐、电影等艺术形式
- 到了如今，便是大语言模型这样的 AI

一本书，便是一段被保存下来的河流。

## 看清你的河

一个人想要真正看清自己，第一步从不是急着追问自己：我贪婪什么？我恐惧什么？

而是先搞清楚—— 你究竟是哪一种河 。

> 有的人宽而浅。有的人窄而深。有的人宽而深，却水流湍急、难以掌控。

水面本就是易动的：风一吹就起波纹，水一多就会泛滥，流速一快就会翻涌。

倘若只盯着水面，便会觉得人生处处是问题；可若是退一步，站在更高的视角看整条河，便会发现许多问题的答案，其实早已写在河流的模样里。

> 「人不能两次踏进同一条河流。」 — 赫拉克利特
